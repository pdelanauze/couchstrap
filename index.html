CouchApp boilerplate

This application skeleton helps reduce couchapp application setup time. The project includes the html5 boilerplate reset, bootstrap (sass version), requirejs, backbonejs, modernizr, webshims, as well as a backbone couch connector.

Quickstart

Clone the project

You'll need the soca gem:
gem install soca

git clone

Update your .couchapprc

Point it to your couchdb instance.

Build

Forget about that crappy setup time, just develop.
